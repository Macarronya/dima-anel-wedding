# Свадебное приглашение — Дима и Анель

Первая рабочая версия мобильного мини-сайта на 5 сентября 2026 года.

## Как открыть сайт

Откройте файл `index.html` двойным кликом. Для публикации сайт можно загрузить на Netlify, GitHub Pages или другой статический хостинг.

## Где менять данные

Основные данные вынесены в файл `config.js`:

- имена;
- дата и время;
- адрес;
- тексты;
- ссылка на карту;
- варианты алкоголя;
- фотография;
- URL Google Apps Script.

Менять HTML для обычных правок не требуется.

## Как добавить вашу фотографию

1. Поместите фотографию в папку `assets`, например `assets/photo.jpg`.
2. В `config.js` замените:

```js
heroImage: "assets/hero-placeholder.svg"
```

на:

```js
heroImage: "assets/photo.jpg"
```

Лучше использовать вертикальную фотографию или фото, которое хорошо кадрируется по центру.

## Как подключить Google Таблицу

1. Создайте новую Google Таблицу.
2. В таблице выберите `Расширения → Apps Script`.
3. Вставьте код из файла `google-apps-script.gs`.
4. Сохраните проект.
5. Запустите функцию `setupHeaders` один раз.
6. Выберите `Развернуть → Новое развертывание`.
7. Тип: `Веб-приложение`.
8. Выполнять от имени: вы.
9. Доступ: все.
10. Скопируйте URL веб-приложения.
11. Вставьте URL в `config.js`:

```js
googleScriptUrl: "ВАШ_URL"
```

После этого ответы гостей будут добавляться в лист `RSVP`.

## Демо-режим

Пока URL Google Apps Script не указан, ответы сохраняются только в Local Storage браузера. Это позволяет проверить форму без подключения таблицы.

## Что уже реализовано

- адаптивный мобильный дизайн;
- главный экран с фотографией;
